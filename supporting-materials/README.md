# Supporting Materials

This folder contains supporting materials for the UTSSRP CCA and contrastive
encoder work. The main encoder comparison experiment lives at the repository
root; these files are earlier notebooks, reference materials, and generated
presentation outputs.

## Contents

- [tapering-cca/](tapering-cca/) contains earlier CCA tapering notebooks and
  spurious-correlation checks.
- [exploratory-notebooks/](exploratory-notebooks/) contains exploratory CCA and
  contrastive-encoder notebooks.
- [presentation-outputs/](presentation-outputs/) contains generated slide decks
  and intermediate presentation artifacts.
- [provided-notebooks/](provided-notebooks/) contains starter/reference
  notebooks and figures provided during the project.
- [reference-materials/](reference-materials/) contains background PDFs and
  slides.
