# Additional Research Materials

This folder keeps supporting material that informed the UTSSRP CCA and
contrastive-encoder project but is not the main contribution path.

For the completed contrastive-encoder contribution, start at the repository
root README, the final summary deck/PDF, or
[../experiment-notebooks/](../experiment-notebooks/). Final report plots and
poster assets live in [../research-outputs/](../research-outputs/).

## Contents

- [tapering-cca/](tapering-cca/) contains earlier CCA tapering notebooks and
  spurious-correlation checks.
- [exploratory-notebooks/](exploratory-notebooks/) contains exploratory CCA and
  contrastive-encoder notebooks from earlier project stages.
- [provided-notebooks/](provided-notebooks/) contains starter/reference
  notebooks and figures provided during the project.
- [reference-materials/](reference-materials/) contains background PDFs and
  slides.
- [presentation-outputs/](presentation-outputs/) contains generated or
  intermediate presentation artifacts from earlier drafts and runtime checks.

These files are kept for traceability and background context. They are useful
for understanding how the project developed, but the final claims should be
checked against the main package, final notebooks, and `research-outputs/`.
